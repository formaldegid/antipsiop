# Автоматизированный классификатор сгенерированных текстов с помощью LLM

## Описание проекта

Проект представляет собой веб-приложения для классификации сгенерированных ИИ текстов при помощи моделей RoBERTa.

## Структура проекта

В репозитории находятся папки **train_model** и **AIGenDetector**. Папка **train_model** содержит ссылку на набор данных для обучения различных моделей "xlm_roberta_ai_detector", блокнот с процессом обучения и созданные модели. Папка **AIGenDetector** содержит готовое web-приложение, позволяющее использовать существующую модель "roberta-base-openai-detector" и обученную модель "xlm_roberta_ai_detector".

## Использование приложения

Начать работу с приложением можно при помощи следующих команд:

```
git clone https://github.com/formaldegid/antipsiop
cd antipsiop/AIGenDetector
```

```
uvicorn app:app --reload
```

После этого перейдите по ссылке: http://127.0.0.1:8000/
